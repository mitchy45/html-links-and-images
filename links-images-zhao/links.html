There are three kinds of links, listed below:

  • Absolute links start with a scheme (like 'https://'), have a domain (like 'developer.mozilla.org'), and a path (like '/en-us/docs/web/html'). This is where that link leads to, in a new window or a new tab, depending on.
  • Relative links are... well, this example relative link leads to our extras page.
  • Root-relative links are kind of like relative links except they're relative to the "root" of the whole site instead of being relative to the current page. Normally, it could go to the home page, but it won't work for these local HTML files.
